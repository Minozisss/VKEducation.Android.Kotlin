## Задание
1. Выполните декомпозицию MVP проекта по требованиям:
   ➔ Список приложений
   ➔ Карточка приложения
   ➔ Установка, удаление приложений

2. Должен получиться список Epic-ов:
   ➔ Каждый не более 2 недель
   ➔ Дайте каждому эпику оценку по времени
   ➔ В каждом эпике - задачи не более дня
   ➔ Дайте каждой задаче оценку по времени


## Feature: Список приложений
### Epic 1.
- Название: Просмотр списка приложений
- Описание: Пользователь может просматривать список всех приложений
- Оценка: 4 дня (27 часов) (7 часов - стандартный рабочий день)

**Tasks:**
- Проанализировать требования к списку приложений - 2 часа
- Продумать структуру списка приложений - 3 часа
- Подготовить дизайн - 2 часа
- Настроить взаимодействие с бекендом по получению списка приложений - 6 часов
- Реализовать (сверстать) экран списка приложений - 4 часа
- Подключить данные к UI - 2 часа
- Реализовать состояние загрузки - 2 часа
- Реализовать состояние ошибки - 2 часа
- Провести тестирование - 4 часа

## Feature: Установка, удаление приложений
### Epic 1.
- Название: Установка приложения
- Описание: Пользователь может установить выбранное приложение
- Оценка: 5 дней (30 часов) (7 часов - стандартный рабочий день) (С запасом)

**Tasks:**
- Проанализировать требования к установке приложения - 2 часа
- Подключить API для установки приложения - 7 часов
- Реализовать взаимодействие с API(отправка запроса на установку) - 7 часов
- Добавить кнопку установки приложения и её состояния - 4 часа 
- Обработать состояние успешной установки - 2 часа
- Обработать состояние ошибки во время установки - 2 часа
- Провести тестирование - 6 часов


### Epic 2.
- Название: Удаление приложения
- Описание: Пользователь может удалить установленное приложение
- Оценка: 4 дня (22 часов) (7 часов - стандартный рабочий день) (С запасом)

**Tasks:**
- Проанализировать требования к удалению приложения - 2 часа
- Подключить API для удаления приложения - 4 часов
- Реализовать взаимодействие с API(удаление приложения с устройства) - 4 часов
- Добавить кнопку удаления приложения и её состояния - 2 часа
- Обработать состояние успешного удаления - 2 часа
- Обработать состояние ошибки во время удаления - 2 часа
- Провести тестирование - 6 часов
